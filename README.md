# 🏭 사일로 투입 오류 방지 시스템

OCR AI 기반 화학 원재료 자동 인식 및 사일로 뚜껑 제어 시스템

## 라인 구성
- **라인 A** (6구): CaCO₃, B₂O₃, ZnO, SiO₂, K₂CO₃, Na₂CO₃
- **라인 B** (9구): B₂O₃, TiO₂, Na₂CO₃, K₂CO₃, SiO₂, CaCO₃, Co₃O₄, Fe₂O₃, CeO₂

## Vercel 배포 방법
1. 이 저장소를 GitHub에 업로드
2. vercel.com에서 GitHub 저장소 연결
3. 환경변수 `REACT_APP_ANTHROPIC_API_KEY` 에 API 키 입력
4. Deploy 클릭
